Cloth Packing Script Documentation

read_file(file_name: str):

This function reads a file containing cloth types, styles, and pieces information. The file should be formatted in a specific way, with each line containing specific information. The function returns a tuple containing the number of cloth types, the number of styles, a dictionary mapping cloth types to compatible styles and vice versa, and a list of pieces.

get_remaining_cloth(pieces):

This function takes a list of pieces and returns a list of pieces that have not been packed into boxes yet.

process_piece(piece, remaining_cloth, compatible, box, cod):

This function processes a piece of cloth, checking its compatibility with other pieces and packing it into a box if possible.

pack_box(number_of_cloth_types: int, number_of_styles: int, compatible: dict[str, list], pieces: list[list]):

This function packs the given pieces of cloth into boxes based on certain criteria. It uses the get_remaining_cloth and process_piece functions to do this. The function returns a tuple containing a list of boxes and a list of remaining cloth pieces that could not be packed into boxes.

main(PATH_TO_FILE: str):

This function reads the file and packs the boxes using the read_file and pack_box functions. It returns a tuple containing the result of read_file and the result of pack_box.

Example:


import cloth_packing_script

PATH_TO_FILE = "cloth_data.txt"
result = cloth_packing_script.main(PATH_TO_FILE)
print(result)

This will read the data from the file "cloth_data.txt", pack the boxes, and print the result.
